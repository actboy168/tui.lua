# API 参考

tui.lua 的 API 按源码架构分为三大环节：

| 环节 | 源码 | 文档 | 说明 |
|------|------|------|------|
| 核心 API | `tui/init.lua` | [core.md](core.md) | 组件、Hooks、工具函数 |
| 高级控件 | `tui/extra/` | [extra.md](extra.md) | TextInput、Select、Spinner 等交互组件 |
| 测试套件 | `tui/testing.lua` | [testing.md](testing.md) | 离屏渲染、输入模拟、快照测试 |
